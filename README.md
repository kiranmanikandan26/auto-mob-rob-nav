### MSc Project: Autonomus Mobile Robot Navigation - SLAM for Wild Environments
### Context Env: Home

## Overview
- This simulation implements basic obstacle avoidance for a home robot
inspired by the GeNIE probing concept (Wang et al., 2025).

- A Python-based simulation of a home robot navigating through 
furniture obstacles to reach a target destination.
